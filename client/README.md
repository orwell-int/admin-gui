# Orwell web client

## Quick notes about development environment

- choco install nvm
- nvm install latest
- nvm use 12.15.0
- choco install yarn

## Running the client

- cd client/admin-app
- yarn
- yarn ng build --prod
- yarn start

## VSCode extensions

- eslint
- prettier
- material icon theme
- debugger for Chrome
- angular snippets
